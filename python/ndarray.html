多次元配列（ndarray型）

配列型のndarray化

配列は、numpy.array()関数で生成できる。

1次元配列を生成する例

1次元配列とはベクトルである。
import numpy as np
a = np.array([1,2,3,4,5])
print(a)
type(a)

出力結果
[1 2 3 4 5]
numpy.ndarray

2次元配列を生成する例

import numpy as np
a = np.array([[1,2,3], [4,5,6]])
print(a)
type(a)

出力結果
[[1 2 3]
 [4 5 6]]
numpy.ndarray

属性意味

配列に関する関数

関数意味
np.zeros(shape)形shapeの要素が全て0の配列を生成
np.ones(shape)形shapeの要素が全て1の配列を生成
