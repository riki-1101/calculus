床関数の積分

床関数とは、\(x\in\mathbb{R}\) に対して、\(x\) を超えない最大の整数を返す関数で、\(\lfloor x \rfloor\) と書かれる。

\(m,n\in\mathbb{Z}\) とするとき \[ \int_m^n\lfloor x \rfloor dx=\sum_{k=m}^{n-1}\int_k^{k+1}\lfloor x \rfloor dx \] ここで、\(k\lt x\lt k+1\) のとき \(\lfloor x \rfloor=k\) であるから \[ \sum_{k=m}^{n-1}\int_k^{k+1}\lfloor x \rfloor dx =\sum_{k=m}^{n-1}\int_k^{k+1}kdx =\sum_{k=m}^{n-1}k\int_k^{k+1}dx =\sum_{k=m}^{n-1}k \] と計算できる。
