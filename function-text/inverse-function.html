逆関数

逆関数の定義

定義（逆関数）
関数 \(f:X\to Y\) が全単射であるとき、関数 \[ f^{-1}:Y\to X \] が定義でき、これを \(f\) の逆関数という。

逆関数の性質

定理（逆関数の性質）
  1. \(f\) と \(f^{-1}\) とでは、定義域と値域が入れ替わる。
  2. \(y=f(x)\) のグラフと \(y=f^{-1}(x)\) のグラフは、直線 \(y=x\) に関して対称である。
